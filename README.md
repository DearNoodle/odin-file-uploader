# Image Storage App

## Key Features

- **User Authentication**
- **File Uploads**
- **Folder Management**
- **File Downloads**
- **File Details**

## Technique Used

- **Backend:**
  - Node.js
  - Express.js (Web Framework)
  - Prisma (ORM for Database Interactions)
  - [Cloudinary] (File Storage)
- **Authentication:**
  - Passport.js
  - Prisma Session Store
- **Upload handling:**
  - Multer
